### \<Button> 과 \<input type='button'> 차이

1. \<Button>
 - 텍스트 뿐 아니라 이미지 or gif 요소를 자식으로 포함 가능
 - 디자인의 폭이 넓다
 - submit의 기능이 default.
    - submit을 하고 싶지 않을경우 type = 'button' 을 속성값으로 명시
 - type 속성 값
    - type = 'submit' 전송
    - type = 'reset' 내용 초기화
    - type = 'button' 기능 구현, javascript 이벤트 실행

2. \<input type='button'>
 - 디자인 컨트롤 제약(css로는 가능)


> * 디자인 적인 제약이 가장 큰차이
